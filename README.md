# Growdash-Hackathon

Team Dinamo

---

#### Local deployment

##### Prerequisites

* git
* Docker
* Docker Compose

##### Run script

```shell
git clone https://github.com/sananeminli/Growdash-Hackathon.git
cd Growdash-Hackathon
docker-compose up --build -d
```

* ### Frontend: http://localhost:5173/
* #### Backend Swagger UI: http://localhost:8000/docs
